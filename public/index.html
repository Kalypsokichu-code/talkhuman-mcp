TalkHuman MCP

AI that writes like a human

What is this

Model Context Protocol server that eliminates AI slop. Based on research analyzing expert annotations from NLP writers and philosophers. arXiv:2509.19163v1

Setup

1. Configure Claude Desktop

Edit your claude_desktop_config.json

{
  "mcpServers": {
    "talkhuman": {
      "url": "https://talkhuman-mcp.vercel.app/api/mcp"
    }
  }
}

2. Restart Claude Desktop

Tools will be available after restart

Tools

get_human_writing_rules

Research-based guidelines for natural writing

check_for_slop

Detect AI patterns in text

get_slop_examples

Examples of patterns to avoid
